# hermes-tweet

Hermes Tweet is a native Hermes Agent plugin for X/Twitter search, social
listening, account reads, and gated account actions.

This Fieldkit package does not vendor the Hermes Tweet runtime. It gives Claude
Code and Codex a concise operator guide for installing the upstream plugin,
choosing read versus action tools, and keeping secrets out of prompts.

Source: <https://github.com/Xquik-dev/hermes-tweet>
