Repository shape

One source. Runtime-specific adapters.

Each plugin lives under packages/<plugin>/. Claude metadata remains the source. Codex packages are generated into .agents/plugins/generated/, so agent-specific runtime shape does not fork the marketplace by hand.

packages/ canonical plugin source
bin/plugin-adapters build, check, and diff generated output
hooks/ repo-level safety and commit discipline
