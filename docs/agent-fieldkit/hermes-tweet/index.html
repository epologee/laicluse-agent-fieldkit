hermes-tweet

Guide Hermes Agent X/Twitter workflows through the source-native Hermes Tweet plugin.

Claude + Codex 2.0.1 1 skill

Hermes Tweet is a native Hermes Agent plugin for X/Twitter search, social listening, account reads, and gated account actions.

This Fieldkit package does not vendor the Hermes Tweet runtime. It gives Claude Code and Codex a concise operator guide for installing the upstream plugin, choosing read versus action tools, and keeping secrets out of prompts.

Source: <https://github.com/Xquik-dev/hermes-tweet>

Install

claude plugins install hermes-tweet@laicluse-agent-fieldkit
codex plugin add hermes-tweet@laicluse-agent-fieldkit
